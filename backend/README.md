# Backend Documentation

## Overview
This document provides an overview of the backend of the Agrico Marketing project.

## Technologies Used
- Node.js
- Express
- MongoDB

## Setup Instructions
1. Clone the repository:
   ```bash
   git clone https://github.com/yasin-yp/Agrico-marketing.git
   ```
2. Navigate to the backend directory:
   ```bash
   cd Agrico-marketing/backend
   ```
3. Install the required dependencies:
   ```bash
   npm install
   ```
4. Start the server:
   ```bash
   npm start
   ```

## API Endpoints
### GET /api/v1/products
- Description: Fetch all products
- Response: List of products

### POST /api/v1/products
- Description: Create a new product
- Request Body: Product details

## Testing
Run the tests using the following command:
```bash
npm test
```
